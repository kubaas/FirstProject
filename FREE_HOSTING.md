# Replit - Free Hosting Setup

Replit oferuje:
- ✅ **Naprawdę darmowe** - bez karty kredytowej
- ✅ Publiczny URL dla aplikacji
- ✅ Java 17 + Maven preinstalowane
- ✅ PostgreSQL w darmowym planie

## Instalacja:

1. Wejdź na https://replit.com
2. Kliknij "Sign Up" → zaloguj się GitHub
3. Kliknij "+ Create" → "Import from GitHub"
4. Wklej: `https://github.com/kubaas/FirstProject`
5. Wybierz "Java" jako język
6. Kliknij "Import"
7. Czekaj ~2 minuty na setup
8. Kliknij "Run" (w górnym menu)

Backend będzie dostępny na:
```
https://FirstProject.kubaas.replit.dev
```

Dla frontendu potrzebujesz osobnego Replit (Node.js).
